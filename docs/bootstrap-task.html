Bootstrap 5 + Luxon

Интеграция модального окна с динамическим временем

Николь Харитошкина · 475224 · Р3269

Описание задания

Задача: встроить приложение с Luxon в шаблон Bootstrap 5.

  • Страница должна содержать 3 колонки (2-8-2).
  • В центральной колонке — большая красная кнопка на всю ширину с текстом «Показать время».
  • При нажатии открывается модальное окно.
  • В заголовке окна — имя и фамилия выполнившего.
  • В теле окна — текущая дата и время через Luxon.
  • Закрытие — крестик и кнопка «Закрыть».

HTML-код страницы

Основные элементы:

  • Подключение Bootstrap 5 через CDN.
  • Подключение Luxon через CDN.
  • Колонки: col-2, col-8, col-2.
  • Модальное окно с data-bs-toggle="modal".
  • Заголовок с именем в .modal-title.
  • Тело с временем, обновляемым через luxon.DateTime.local().
  • Кнопка закрытия с data-bs-dismiss="modal".
