Time задание после теста

1.Как вы думаете откуда береться переменная? Какое у нее значение?

Х является глобальной переменной, переменная хранит начальный момент времени в миллисекундах (Unix timestamp). Она используется для расчета прошедшего времени. Date.now() - текущее время в момент каждой итерации.

2. Попросите переписать это выражение без использования IIFE и удостоверьтесь, что работает этот вариант аналогично.

без IIFE:

console.time('a');

y = 2 ** 15;

x = Date.now();

while (Date.now() - x < y) ;

console.timeEnd('a');

IIFE - это функция которая выполняется сразу после ее создания, без необходимости отдельного вызова
