Описание задания (Этап 3)

Задача: реализовать приложение с Bootstrap и Luxon, используя сборщик Vite.

  • Создать проект с Vite.
  • Интегрировать Bootstrap и Luxon.
  • Реализовать три колонки (2-8-2), кнопку и модальное окно.
  • Минимизировать размер бандла.
