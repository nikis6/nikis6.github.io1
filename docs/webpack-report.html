Webpack Docker Bootstrap

Инициализация проекта

1Создание папки и инициализация
mkdir project1
cd project1
npm init -y

Создана папка проекта

2Установка зависимостей
npm i luxon
npm i -D webpack webpack-cli serve

Установлен luxon для работы с датой/временем, webpack и webpack-cli для сборки.

Настройка Webpack

1Создание webpack.config.js
const path = require('path'); module.exports = { entry: './src/index.js', output: { path: path.resolve(__dirname, 'dist'), filename: 'main.js' }, mode: 'development' };

Точка входа — src/index.js, результат — dist/main.js.

Код с Luxon

1Создание src/index.js
const { DateTime } = require('luxon'); const hh = document.createElement('h1'); document.body.appendChild(hh); setInterval(() => { hh.textContent = DateTime.local().setLocale('ru').toFormat('dd.Ll.Y HH:mm:ss'); }, 1000);

Код создаёт элемент <h1> и обновляет время каждую секунду.

Сборка проекта

1Команда сборки
npx webpack

Результат: compiled successfully — файл dist/main.js (включает код + luxon).

Работа с Docker

1Dockerfile
FROM node:24-alpine WORKDIR /app COPY package*.json ./ RUN npm install COPY . . RUN npx webpack EXPOSE 8080 RUN npm install -g serve CMD ["serve", "-s", ".", "-l", "8080"]

Используется образ node:24-alpine для минимального размера.

2Сборка образа
docker build -t my-webpack-app .

Создан образ my-webpack-app.
